This is a simple and small plugin to customize your mouse cursor and social media.

== Description ==

Festive Cursor Trail WordPress Plugin is developed by Akestech Infotech to decorate your blog with new cursor trail.  This will add a custom text trial for Every kind of Fest. And have extra feature to add social icon on footer of every post. Most of the bloggers or non bloggers wants to decorate their blog on festive season like New Year, Christmas, Halloween, Diwali, Holi, Eid..etc. They may use this plugin for decoration, It also provide option of Share Social Media buttons and Decorative Lights.
Belongs to any country, this plugin doesn't separate the peoples, just type your text to show trail on mouse, Enable disable plugin for your blog.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/festive-cursor-trail` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Festive Cursor Trail Option Menu Left side of screen to configure the plugin
